# umlify.com

Website: http://umlify.com

## Description
Umlify.com allows Users to upload multiple Java source files and download generated UML Class Diagrams as either SVGs or PNGs.


## Stack
+ React + Redux
+ Spring Boot 2
+ Java Parser Library
+ Docker/Docker-compose + NginX
+ Digital Ocean Droplet
